# Kali Linux
```
$ sudo apt install git
$ git clone https://github.com/katandev18/wifiphiser
$ cd wifiphiser
$ chmod +x install.sh
$ ./install.sh <os type>
$ ./wifiphiser.sh
> checking os type dpkg --print-architecture
> 64bit: amd64
> 32bit: i386
```
# Termux
```
$ pkg install git
$ git clone https://github.com/katandev18/wifiphiser
$ cd wifiphiser
$ chmod +x install.sh
$ ./install.sh <os type>
> checking os type dpkg --print-architecture
> 64bit: arm64
> 32bit: arm
```
